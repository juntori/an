# 1차 딥리서치 — 한국 유아·초등 저학년 교육 시장

- 조사일: 2026-07-06
- 방법: 딥리서치 워크플로 (103개 에이전트, 21개 출처, 96개 주장 추출 → 상위 25개 3표 교차검증 → 22개 확정, 3개 기각)
- 질문: 무자본 1인 창업자가 디지털 콘텐츠로 진입하기에 가장 유망한 교육 주제는?

## 핵심 결과

### 시장 전체

| 지표 | 수치 | 신뢰도 |
|---|---|---|
| 2025년 사교육비 총액 | 27.5조 원 (-5.7%, 5년 만에 첫 감소) | high |
| 참여 학생 1인당 지출 | 월 60.4만 원 (+2.0%, **사상 최고**) — 골드키즈 정량 확인 | high |
| 초등 세그먼트 | 총액 12.2조 원, 참여율 84.4% (학교급 중 최대·최고) | high |
| 유아 참여율 | 만 3세 50.3% → 만 4세 68.9% → **만 5세 81.2%** | high |
| 유아 참여 아동 지출 | 월 33.2만 원 (2024 시험조사) | high |

**해석**: 총량은 축소 국면이나 참여 아동 1인당 지출은 역대 최고. "밀도의 시장" — 프리미엄·개인화 상품에 유리.

### 과목별

| 과목 | 전체 학생 기준 | 참여 학생 기준 |
|---|---|---|
| 영어 | 월 13.1만 원 (1위) | 월 28.1만 원 |
| 수학 | 월 12.8만 원 | — |
| 국어 | 월 3.9만 원 | 월 18.5만 원 |
| 사회·과학 | 월 1.9만 원 | 월 16.6만 원 |

- 영어·수학과 국어의 3~7배 격차는 **참여율 차이**가 주도 — 참여자 기준으로는 1.5~1.7배로 좁혀짐 → 국어(문해력)는 "덜 포화된" 니치
- 유아 영어는 참여 아동 기준 **월 41.4만 원**으로 전 과목 압도적 1위 (지불 의사 최고)

### 경쟁 신호

- 인쇄 학습지 PDF 니치에 법인 사업자 존재: '참잘했어요'(㈜예스폼) — 한글·수학·영어 다과목 번들 구독
- PDF 콘텐츠 부업 수익은 극단 분포 (상위 억대, 대다수 월 10만 원 미만) — **성패는 SNS 채널 선구축**
- ⚠️ 참잘했어요 월 22,000원 가격 클레임은 검증 기각(0-3) — 가격 벤치마크로 사용 금지

### 규제

- PDF/온라인 콘텐츠 판매: 사업자등록 + 통신판매업 신고
- 교구/키트 확장 시: KC 어린이제품 안전인증 (모델별 시험·검사 비용) — 소프트웨어는 비대상
- 만 14세 미만 아동 개인정보: 법정대리인 동의 필수

## 최종 추천 (리서치 시점)

1. **1순위 한글/문해력** — 덜 포화, 유아→초1 자연 연결, 비전문가 제작 리스크 최소
2. **2순위 수학/사고력** — 초등 최대 지갑 점유율, 안정 수요

→ 이후 창업자 아이디어(웹캠 인터랙션 게임 학습)와 결합하며 **영어 우선**으로 방향 수정. 인터랙션이라는 차별화 축이 영어의 높은 경쟁 강도를 상쇄하고, 지불 의사 1위 과목의 이점을 취하는 전략. 한글은 확장 라인으로 유지.

## 데이터 한계 (caveats)

1. 2025년 조사가 최신 — 총액 감소가 일시적인지 구조적인지는 2026년 조사로 재확인 필요
2. 초중고 조사는 초등 1~6학년 전체 기준, 유아 조사는 6세 미만 기준 — 6~9세만 분리한 통계 없음
3. 유아 통계는 국가 미승인 '시험조사'(2024년 7~9월 분기) — 정식 승인 통계는 2026년부터
4. 코딩/AI·과학·경제교육 니치의 시장 규모, 대형 학습지 기업(웅진·교원·대교) 세부 경쟁 데이터는 검증 통과 출처 없음 — 공백
5. 경쟁 판단은 단일 사업자 관찰 기반 — 니치 전체 포화도로 일반화 금지
