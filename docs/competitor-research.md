# 2차 딥리서치 — 유아 영어 에듀테크 경쟁 지형

- 조사일: 2026-07-06
- 방법: 딥리서치 워크플로 (105개 에이전트, 3표 교차검증)
- 질문: "웹캠 동작 인식 + 음성 인식 브라우저 영어 게임"의 차별화 공간이 실재하는가? 가장 위협적인 경쟁자와 검증된 가격 벤치마크는?

## 결론

**차별화 공간은 실재하나 조건부.** "웹캠 동작인식 + 아동 음성 + 브라우저 + 가정용 B2C 구독"의 정확한 조합을 가진 상용 제품은 확인되지 않았다. 다만 인접 공간은 점유되어 있다:

- **B2B 교실**: 비상교육 '챌린지' Action Alphabet — 모션인식(MR) 기반 유아 알파벳, 전자칠판/웹캠 PC, 유치원 채널 상용화 (가장 직접적인 컨셉 경쟁자)
- **학술 선례**: MS Research SpatialEase (CHI 2013, Kinect TPR 게임) — 개념을 선점했으나 연구 프로토타입
- **B2C 저가 물량**: 링고키즈 — 터치 기반이나 연령대가 직접 겹침

## 검증된 가격 벤치마크 (2026-07-06 스냅샷)

| 서비스 | 상품 구조 | 검증 가격 | 신뢰도 |
|---|---|---|---|
| 토도영어 | 자녀 수 기준 기간제 | 12개월 528,000원 (실효 월 44,000원), 6개월 299,000원, 애플 IAP 월 58,500원 | high (결제 API 직접 확인) |
| 링고키즈 | 월/연 구독, 2~8세 | 월 23,500원, 연 112,000~139,000원 | high (KR 앱스토어 IAP) |
| 호두잉글리시 | 500일권 + 갤럭시탭 번들 | 태블릿(정가 45만원) 결합 고가 패키지, 이용권 실판매가 주장은 검증 기각 | 번들 전략만 high |

→ **B2C 포지셔닝 밴드: 월 23,500 ~ 44,000원.** 신규 브랜드는 하단 아래(월 14,900~19,900원)로 침투 후 카탈로그 확충과 함께 인상.

## 기술 리스크: 유아 발화 인식 (최대 리스크 확정)

- 2025년 JASA 연구: 최신 Siri/Alexa도 2~5세 발화 전사에 실패. 인간 청자가 전 연령에서 기계를 크게 앞섬
- 비원어민 아동 자유발화 WER 17.14% vs 낭독체 2.26% (약 7.6배) — **자유 발화 인식은 신뢰 불가**
- 업계 표준 아동 ASR이던 **SoapBox Labs는 Curriculum Associates에 인수(2023 말)되어 외부 라이선스 사실상 불가** — "사서 쓰면 된다"는 전제 기각(0-3)
- **설계 결론: 프롬프트형(캐릭터가 먼저 말한 단어 따라 말하기)으로 제한.** 장기적으로 Whisper류 파인튜닝(Kid-Whisper 연구 경로) 검토

## TPR 디지털화의 교육학 근거

- SpatialEase 파일럿: 동작 기반 학습이 기존 시각 학습법(Rosetta Stone)과 동등한 어휘 성과
- ⚠️ 성인 8명, 단일 세션 파일럿 — 4~9세 효능은 자체 베타로 입증해야 함. 체화학습(embodied learning) 메타분석이 방향성 보강

## B2B 채널 규제 (어린이집 vs 유치원)

**어린이집 특별활동 (영유아보육법, 3-0 검증):**
- 외부 강사 운영이 법정 요건 → 소프트웨어 단독 라이선스 모델 부정합
- 시·도지사 수납한도 상한: 서울 월 110,000원(2025), 경기 90,000원(2026)
- 월 단위 과금 의무, 학부모→업체 직접 지불 금지(원 계좌/아이행복카드 경유)
- 공급 구조는 강사 파견 지사 체계, 참여 아동 인당 과금이 주류

**→ 전략: 어린이집은 특별활동 파견업체에 콘텐츠 공급하는 제휴 모델, 직접 라이선스는 유치원(교육청 관할)·영어학원에 집중.**

## 데이터 한계 (caveats)

1. 학부모 선택 기준·스크린타임 우려 소비 트렌드는 검증 통과 근거 전무 — "움직이는 스크린타임" 포지셔닝은 미검증 가설, 베타 인터뷰로 확인 필요
2. 야나두 키즈·윙크·뽀로로 영어의 실가격 미확인 — B2C 벤치마크는 토도·링고키즈 2축 의존
3. 특별활동 참여율·비용 데이터는 2012~2013년 KICCE 조사 (하한 참고치로만 사용); 수납한도는 2025~2026 현행
4. 아동 ASR 연구는 Siri/Alexa·구세대 모델 기준 — 최신 Whisper급의 한국 아동 영어 발화 성능은 미검증 (Week 1 기술 스파이크에서 직접 확인)
5. 가격은 2026-07-06 스냅샷, 프로모션 변동 가능

## 미해결 질문

1. 비상교육 챌린지의 실제 B2B 단가·보급 원 수, B2C 확장 계획
2. Whisper급 파인튜닝 시 한국 아동 영어 프롬프트형 발화 인식률이 게임 UX 수준(90%+)에 도달 가능한가
3. 유치원·영어학원 채널의 외부 디지털 콘텐츠 도입 구조·단가 (어린이집보다 규제 부담이 낮을 가능성)
