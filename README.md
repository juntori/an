# 몸으로 배우는 영어 (가칭)

웹캠 동작 인식 + 음성 인식 기반 유아 영어 게임 학습 서비스.

아이가 화면 속 사과를 손으로 따는 동작을 하면 웹캠(MediaPipe)이 인식하고, "Apple!"이라고 말하면 음성이 인식되어 이야기가 진행되는 **브라우저 기반 인터랙션 영어 학습**입니다. 검증된 유아 영어 교수법 TPR(전신반응 교수법)의 디지털 구현입니다.

## 핵심 컨셉

- **타겟**: 4~9세 (유아~초등 저학년), 영어 우선 → 한글/문해력 확장
- **형태**: 브라우저 웹앱 (노트북/태블릿, 앱 설치 불필요)
- **기술**: MediaPipe Hands(동작) + Web Speech API(음성) — 전부 클라이언트 사이드, 카메라 영상 미저장·미전송
- **수익**: B2C 월 구독 + B2B 기관 라이선스(유치원/어학원) + 맞춤 동화책 굿즈(POD)
- **조건**: 무자본 1인 전업 창업, AI 활용 콘텐츠 직접 제작
